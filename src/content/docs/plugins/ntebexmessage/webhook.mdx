---
title: Discord webhook
description: Send purchase notifications to Discord using webhook embeds.
---

NTebexMessage can post a Discord webhook every time a template is broadcasted, so your community sees live store activity without being in-game.

Configuration file:

```
discord.yml
```

For full embed field reference, see the shared [Discord webhook builder](/common/discord-webhook-builder/) page.

## Enable

```yaml
webhook:
  enabled: true
  url: "INSERT-WEBHOOK-URL"
```

| Field | Description |
| --- | --- |
| `enabled` | `true` to send a webhook for every broadcast. |
| `url` | The Discord webhook URL. |

## Bot identity

```yaml
username: "Store Notification"
avatar-url: "https://i.imgur.com/ip1xV13.png"
```

| Field | Description |
| --- | --- |
| `username` | Display name of the webhook bot. |
| `avatar-url` | Image shown as the bot avatar. |

## Multi-package separator

When a purchase contains multiple packages, the separator joins them in the Discord message.

```yaml
symbol: "•"
```

Input: `VIP;MVP;ASTRO` → output: `VIP • MVP • ASTRO`

## Cleaning package names

### Word filter

Remove unwanted words from package names before they reach Discord. Case-insensitive.

```yaml
word-does-appear:
  - kit
  - rank
  - package
```

Input `Kit ASTRO` becomes `ASTRO`.

### Word replace

Replace package names with custom display names.

```yaml
word-replace:
  vip: "VIP"
  mvp: "MVP"
  astro: "ASTRO"
```

## Simple content message

The plain Discord message — useful for mentions.

```yaml
content: "Player **{player}** purchased **{package}**"
```

| Placeholder | Replaces with |
| --- | --- |
| `{player}` | Buyer name |
| `{package}` | Purchased package(s) |
| `{price}` | Purchase price |

## Embed

Embeds are configured under `embed:` and follow the [shared embed format](/common/discord-webhook-builder/).

```yaml
embed:
  enabled: true

  title: "🛒 New Purchase"

  color: "#00FFAA"

  description:
    - "Player **{player}** purchased **{package}**"

  fields:

    - name: "Player"
      value: "{player}"
      inline: true

    - name: "Package"
      value: "{package}"
      inline: true

    - name: "Price"
      value: "{price}"
      inline: true

  author:
    name: "Server Store"
    url: ""
    icon: ""

  footer:
    text: "Powered by Nautic Studios"
    icon: ""

  image: "https://i.imgur.com/ip1xV13.png"

  thumbnail: "https://mc-heads.net/avatar/{player}"
```

| Field | Description |
| --- | --- |
| `title` | Title shown at the top of the embed. |
| `color` | Vertical bar color, HEX. |
| `description` | List of lines rendered as the embed description. |
| `fields` | Structured fields shown inside the embed. `inline: true` puts fields side by side. |
| `image` | Large image at the bottom of the embed. |
| `thumbnail` | Small image at the top-right. Use `mc-heads.net/avatar/{player}` to show the buyer's head. |
