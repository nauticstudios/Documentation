Welcome to the official **Nautic Studios** documentation. This site is the single source of truth for every plugin, library and API maintained by the studio. All content is versioned alongside the source code, so what you read here matches what ships.

Nautic Studios builds production-grade tools for Minecraft servers: lightweight, async by default, and designed to play nicely with other plugins. You'll find both end-user documentation for our plugins and developer references for the libraries and APIs you can build on top of.

:::note
If a feature is not documented on this site, treat it as unofficial until this page is updated.
:::

## Getting help

If something is unclear or behaves unexpectedly:

1. Read the relevant plugin, library or API page on this site.
2. Compare your configuration against the examples shown on the page.
3. Ask on the official Discord if the issue persists — staff and community devs are active there.
