This site is the official source of documentation for every plugin and shared system developed by Nautic Studios.

:::note
All content here is maintained alongside the source code. If a feature is not documented, treat it as unofficial until this page is updated.
:::

## What you will find here

- **Common features** — systems used across every plugin, including the multilingual engine, supported color formats and Discord webhook embeds.
- **Plugins** — full documentation for every released plugin, with overview, commands, configuration and feature pages.
- **Work in progress** — upcoming plugins that have been announced but are not yet available.

## Getting help

If something is unclear or behaves unexpectedly:

1. Read the relevant plugin page.
2. Compare your configuration against the examples shown on the page.
3. Ask on the official Discord if the issue persists.
