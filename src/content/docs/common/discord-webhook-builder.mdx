description: Reference for the shared Discord embed message format.

This page documents the shared Discord webhook embed format used across Nautic Studios plugins. The exact fields available depend on the plugin, but the schema is consistent.

:::caution
Some embed features may not be supported by every plugin. Check the plugin's own documentation for the fields it accepts.
:::

## Basic example

```yaml
content: "Get ready!"
color: "#FF0000"

title:
  text: "Example - STARTED"

fields:
  "1":
    inline: true
    name: "Time"
    value: "100"
```

## Content

The `content` field sends a plain Discord message. It is the **only field that supports working mentions**.

| Mention type | Format |
| --- | --- |
| Role | `<@&1234567890123456789>` |
| User | `<@1234567890123456789>` |
| Channel | `<#1234567890123456789>` |

```yaml
content: "Content field"
```

## Embed fields

### Title

```yaml
title:
  text: "Title text"
  icon: ""           # optional
```

`text` is required. `icon` is optional and is displayed next to the title.

### Color

```yaml
color: "#RRGGBB"
```

Defines the vertical bar on the left side of the embed. Must be a valid HEX color.

### Description

```yaml
description: "Description text"
```

Used for extended information inside the embed body.

### Fields

Multiple fields can be defined using indexed keys.

```yaml
fields:
  "1":
    name: "Name"
    value: "Value"
    inline: true

  "2":
    name: "Another field"
    value: "Another value"
    inline: false
```

| Property | Description |
| --- | --- |
| `name` | Field title — required |
| `value` | Field content |
| `inline` | If `true`, fields appear side by side |

### Author

```yaml
author:
  name: "Author text"
  icon-url: "https://example.com/icon.png"
  url: "https://example.com"
```

| Property | Description |
| --- | --- |
| `name` | Author name — required if the section is used |
| `icon-url` | Optional author avatar |
| `url` | Optional clickable link |

### Footer

```yaml
footer:
  text: "Footer text"
  icon: "https://example.com/footer.png"
```

`text` is the footer message. `icon` is optional.

### Image

```yaml
image-url: "https://example.com/image.png"
```

Displays a large image inside the embed.

### Thumbnail

```yaml
thumbnail-url: "https://example.com/thumbnail.png"
```

Displays a small image on the right side of the embed.
