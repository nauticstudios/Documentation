Main configuration file:

```
config.yml
```

---

## Plugin Settings

```yaml
plugin:
  debug: false
  auto-save-on-refill: true
  log-refill-errors: true
```

---

## Refill Settings

```yaml
settings:
  default-refill-seconds: 300
  target-distance: 6
  clear-before-refill: true
  refill-only-if-empty: false
  max-items-per-refill: 27
  randomize-slot-order: true
```

---

## Broadcast

```yaml
settings:
  broadcast-refill: false
  broadcast-permission: ""
  broadcast-message:
```

Placeholders:
- %world%
- %x%
- %y%
