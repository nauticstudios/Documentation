Configuration file: `discord.yml`. Two modes: `classic` (embed + buttons) or `components-v2` (native container layout).

## Setup

```yaml title="discord.yml"
webhook:
  enabled: true
  url: "INSERT-WEBHOOK-URL"
  username: "Nautic Studios"
  avatar: "https://i.imgur.com/ip1xV13.png"

  mode: components-v2   # or: classic
```

## Filters

Applied **only to substituted placeholders** (`{player}`, `{package}`, â€¦). Literal text is untouched.

```yaml title="discord.yml"
filters:
  strip-words:
    - kit
    - rank
    - package

  replace:
    vip: "VIP"
    mvp: "MVP"

  multi-package:
    multiline: true
    bullet: "â€˘ "
    inline-separator: " â€˘ "
```

| Filter | Behavior |
| --- | --- |
| `strip-words` | Removes the listed words (case-insensitive). `Kit ASTRO` â†’ `ASTRO`. |
| `replace` | Maps raw input â†’ display string. |
| `multi-package.multiline` | `true` = one per line with `bullet`. `false` = inline with `inline-separator`. |

## Placeholders

| Placeholder | Replaces with |
| --- | --- |
| `{player}` | Buyer name |
| `{package}` | Purchased package(s) |
| `{price}` | Purchase price |
| Any `customs-args` key | Its value |

## Mode: classic

```yaml title="discord.yml"
classic:
  content: ""

  embed:
    enabled: true
    title: "đź›’ New Purchase"
    url: ""
    color: "#00FFAA"

    description:
      - "**{player}** purchased **{package}**"

    fields:
      - { name: "Player",  value: "{player}",  inline: true }
      - { name: "Package", value: "{package}", inline: true }
      - { name: "Price",   value: "{price}",   inline: true }

    author:    { name: "Server Store", url: "", icon: "" }
    footer:    { text: "Powered by Nautic Studios", icon: "" }
    image:     ""
    thumbnail: "https://mc-heads.net/avatar/{player}"

  buttons:
    - { label: "Visit Store", url: "https://store.example.com" }
    - { label: "Join Discord", url: "https://discord.gg/example" }
```

| Field | Description |
| --- | --- |
| `content` | Plain text above the embed (mentions go here). |
| `description` | List of lines. |
| `fields[].inline` | `true` lays fields side by side. |
| `thumbnail` | Top-right image. Use `mc-heads.net/avatar/{player}` for the buyer's head. |
| `buttons` | Up to **5** link buttons. |

## Mode: components-v2

```yaml title="discord.yml"
components-v2:
  accent: "#00FFAA"

  layout:

    - type: text
      content: "## đź›’ New Purchase"

    - type: separator

    - type: section
      text:
        - "**Player:** `{player}`"
        - "**Package:** `{package}`"
        - "**Price:** `{price}`"
      thumbnail: "https://mc-heads.net/avatar/{player}/128"

    - type: separator
      spacing: 2

    - type: text
      content: "-# Powered by Nautic Studios"

    - type: buttons
      items:
        - { label: "Visit Store",  url: "https://store.example.com" }
        - { label: "Join Discord", url: "https://discord.gg/example" }
```

`accent` colors the left bar and wraps the layout in a Discord Container.

### Node types

| `type` | Fields |
| --- | --- |
| `text` | `content`: string or list of strings. Markdown supported. |
| `separator` | `divider` (`true` / `false`, default `true`), `spacing` (`1` / `2`). |
| `section` | `text` (list), `thumbnail` (image URL). |
| `media` | `items: ["url", ...]` â€” image gallery. |
| `buttons` | `items: [{ label, url }]` â€” max 5 per row. |
