Welcome to the official **Nautic Studios** documentation.

This site contains verified and up-to-date information for every plugin and shared feature developed by Nautic Studios. Each section is maintained alongside the source code so that what you read here matches what is shipped.

:::note
All content published here is considered authoritative. If a feature is not documented, it should not be considered officially supported until this site is updated.
:::

## How this documentation is organised

- **Getting Started** — what you are reading now. Orientation and support.
- **Common Features** — systems shared across every plugin (languages, colors, webhook embeds, item builder).
- **Plugins** — one section per plugin with overview, commands, configuration and feature pages.

## Reporting Issues

If you experience unexpected behavior or need clarification:

1. Read the relevant plugin documentation.
2. Check the configuration examples on the page.
3. If the issue persists, contact support.
