title: Colors
description: Supported color and text formatting formats across all Nautic Studios plugins.
---

Every Nautic Studios plugin supports the same set of color and text formatting systems. **MiniMessage is the recommended format** — it is the most expressive and works in every module.

## Supported formats

| Name | Example | Generator |
| --- | --- | --- |
| MiniMessage | `<red>Text</red>` | [MiniMessage Web UI](https://webui.advntr.dev/) |
| HEX | `&#RRGGBB` | [HEX Generator](https://birdflop.com/resources/rgb/) |
| Legacy | `&x&R&R&G&G&B&B` | [Legacy Converter](https://birdflop.com/resources/rgb/) |

## MiniMessage

Use MiniMessage for all new configurations. It supports named colors, HEX values, gradients, bold, italic, underline and strikethrough.

```
<gradient:#00FFAA:#00AAFF><bold>Example Text</bold></gradient>
```

:::note
Only color and basic text formatting are guaranteed in every module. Advanced features such as click events or custom fonts may not be supported everywhere.
:::

## HEX

Modern hex colors use the `&#RRGGBB` format:

```
&#00FFAAExample Text
```

## Legacy

Legacy formatting is kept for backwards compatibility:

```
&x&0&0&F&F&A&AExample
```

:::caution
The legacy format is not recommended for new configurations. Use MiniMessage instead.
:::
