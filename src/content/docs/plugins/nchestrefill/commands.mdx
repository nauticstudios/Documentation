## Commands

| Command | Permission | Description | Default |
|----------|------------|-------------|----------|
| `/nchestrefill help` | `nchestrefill.help` | View help menu | true |
| `/nchestrefill editor` | `nchestrefill.editor` | Open content editor GUI | OP |
| `/nchestrefill save` | `nchestrefill.save` | Save held item to default content pool | OP |
| `/nchestrefill addchest` | `nchestrefill.addchest` | Register the chest you are looking at | OP |
| `/nchestrefill removechest` | `nchestrefill.removechest` | Remove the chest you are looking at | OP |
| `/nchestrefill view` | `nchestrefill.view` | Toggle particle beam view mode | OP |
| `/nchestrefill list` | `nchestrefill.list` | List all registered chests | OP |
| `/nchestrefill tp <id>` | `nchestrefill.tp` | Teleport to registered chest by ID | OP |
| `/nchestrefill sync` | `nchestrefill.sync` | Synchronize all chest refill timers | OP |
| `/nchestrefill forcerefill` | `nchestrefill.forcerefill` | Instantly refill all chests | OP |
| `/nchestrefill settime <seconds>` | `nchestrefill.settime` | Set global refill interval | OP |
| `/nchestrefill reload` | `nchestrefill.reload` | Reload plugin configuration and data | OP |

---

## Admin Permission

| Permission | Description | Default |
|------------|-------------|----------|
| `nchestrefill.admin` | Grants access to all commands and bypasses permission checks | OP |
