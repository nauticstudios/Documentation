NHubPvP is a controlled Hub PvP system designed to manage combat activation, kits, abilities and persistent statistics in lobby environments.

---

## Core Features

- Hold-to-activate PvP system
- Configurable activation and deactivation countdown
- Automatic kit application and removal
- Configurable sword slot enforcement
- Optional shield support
- Built-in right-click ability system
- Instant heal on kill (toggleable)
- Persistent kills and deaths tracking
- Global top rankings system
- In-game editor GUI
- PlaceholderAPI support
- H2 and MySQL database support
- Safe runtime reload system

---

## How It Works

1. Player holds the configured sword.
2. Activation countdown starts.
3. Full PvP kit is automatically applied.
4. Player can fight and use ability.
5. Switching away from the sword triggers deactivation countdown.
6. Kit is removed automatically when PvP is disabled.

All activation handling and protection logic is internally managed.
