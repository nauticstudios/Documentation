Three files under `/plugins/NTebexMessage/`.

| File | Purpose |
| --- | --- |
| `config.yml` | Custom args, templates, buttons, reward event. |
| `messages.yml` | UI strings — help, args, errors, console output. |
| `discord.yml` | Webhook bot identity, filters, classic / Components V2 layout. |

Reload after editing any of them:

```
/ntb reload
```

:::tip
Reload is hot — no need to restart the server. Templates, buttons and webhook layouts are re-read in place.
:::

## `config.yml`

```yaml
customs-args: []
separator-symbol: "&7•"

broadcast:
  templates:
    one:        { … }
    multiple:   { … }

buttons:
  center: 20
  list:
    shop:     { … }
    discord:  { … }

reward-event:
  words: []
  players-write-word: 30
  ignore-case: true
  cooldown:                { enabled, time }
  bossbar:                 { enabled, text, color, style }
  actionbar:               { enabled, text }
  null-players: "..."
  players-rewards-commands: []
  final-message:           { enabled, message }
```

## `messages.yml`

```yaml
help:    [ ... ]
args:    { header, empty, format, footer }
console: { usage, invalid-format, unknown-arg, available-args, missing-parenthesis }

reload: "..."
no-permission: "..."
missing-arg: "..."
console-only: "..."
```

| Placeholder | Replaces with |
| --- | --- |
| `{command}` | The alias the player used. |
| `{arg}` | Offending argument name. |
| `{args}` | Available custom arguments. |
| `{usage}` | Auto-built usage line. |

## `discord.yml`

Full reference on the [Discord webhook](/plugins/ntebexmessage/webhook/) page.

```yaml
webhook:
  enabled: false
  url: "INSERT-WEBHOOK-URL"
  username: "Nautic Studios"
  avatar: "..."
  mode: components-v2   # or: classic

  filters:       { strip-words, replace, multi-package }
  classic:       { content, embed, buttons }
  components-v2: { accent, layout }
```

:::danger
`discord.yml` holds your webhook URL. Anyone with that URL can post in the channel. Keep it out of public repos and rotate it if exposed.
:::

## Permissions

| Permission | Description |
| --- | --- |
| `tebexmessage.help` | `/ntb help`. |
| `tebexmessage.reload` | Reload config. |
| `tebexmessage.args` | List custom arguments. |
| `tebexmessage.admin` | Bypasses every check. |
