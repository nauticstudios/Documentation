## Commands

| Command | Permission | Description | Default |
|----------|------------|-------------|----------|
| `/nhubpvp` | — | Displays plugin information and version | true |
| `/nhubpvp help` | `nhubpvp.help` | View help menu | OP |
| `/nhubpvp reload` | `nhubpvp.reload` | Reload configuration, items, messages and menus | OP |
| `/nhubpvp settings` | `nhubpvp.settings` | Open the in-game editor GUI | OP |

---

## Admin Permission

| Permission | Description | Default |
|------------|-------------|----------|
| `nhubpvp.admin` | Grants access to all commands and bypasses permission checks | OP |
