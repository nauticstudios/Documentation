Turns Tebex store purchases into rich in-game broadcasts driven by `config.yml`.

| | |
| --- | --- |
| **Base command** | `/ntb` |
| **Aliases** | `ntebex`, `ntebexmessage`, `tb`, `tbmsg`, `tbxm`, `tbxmsg`, `tebexmessage` |
| **API** | Bukkit `1.16+`, Java `17+` |
| **Folia** | Supported |
| **Soft-depend** | PlaceholderAPI |

## Example

```
/ntb one player(Senkex) package(VIP) price($10)
```

Runs the `one` template, substitutes `{player}`, `{package}`, `{price}` and dispatches whatever the template has enabled — chat, title, sound, actionbar, bossbar, buttons, reward event, Discord webhook and the buyer's head.
