NChestRefill is a structured chest refill system designed for controlled loot management.

---

## Core Features

- Register refillable chests
- Configurable refill intervals
- Paginated content editor GUI
- Per-item drop chance system
- Countdown holograms
- Particle-based view mode
- Runtime-safe reload
- Performance-controlled processing

---

## How It Works

1. Register a chest using `/nchestrefill addchest`
2. Configure items inside the Editor
3. Set refill interval
4. Chest refills automatically when timer reaches zero

All timing and processing is internally managed.
